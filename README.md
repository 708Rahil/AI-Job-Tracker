# AI Job Tracker

**AI Job Tracker** is a full-stack application to manage your job applications with AI-powered summaries.  
Paste a job posting’s title, company, and description, and the AI will generate a concise summary. Track your application status easily!

---

## **Features**

- Add jobs with **title, company, and full description**  
- **AI-generated summaries** using FastAPI + Transformers  
- Track status: **Applied, Interviewing, Rejected, Offer**  
- Save all jobs in **PostgreSQL**  
- React frontend for **easy interaction**  
- Filter jobs by status (future feature)  

---

## **Tech Stack**

- **Frontend:** React  
- **Backend:** Node.js, Express, Axios  
- **AI Service:** Python, FastAPI, Hugging Face Transformers  
- **Database:** PostgreSQL  
- **Other Tools:** CORS, Docker (optional)
